ROS Workspace Setup and Package Creation Steps

出现的小乌龟命令:

roscore

rosrun turtlesim turtlesim_node

rosrun turtlesim turtle_teleop_key

  1. echo $ROS_PACKAGE_PATH
  2. mkdir -p ~/catkin_ws/src
  3. chmod -R 777 ~/catkin_ws
  4. chmod -R 777 ~/catkin_ws/src
  5. catkin_init_workspace
  6. cd ~/catkin_ws/
  7. catkin_make
  8. echo "source ~/catkin_ws/devel/setup.bash" >> ~/.bashrc
  9. source ~/.bashrc
  10. cd ~/catkin_ws/src
  11. catkin_create_pkg test std_msgs rospy roscpp
  12. cd ~/catkin_ws/
  13. catkin_make
  14. cd ~/catkin_ws/src
  15. catkin_create_pkg turtle_vel_ctrl roscpp geometry_msgs

注意：当不够权限的时候在命令前添加sudo

命令来源于书本26~37
